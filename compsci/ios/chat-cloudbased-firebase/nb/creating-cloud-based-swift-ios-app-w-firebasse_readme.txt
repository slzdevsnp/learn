ch: Connecting your app to Firebase

    a plist file from firebase will be added to the xcode project

	start registering  on  firebase.google.com (szldev)
    go to docs  ios -> getting started guide
    go to console in a new tab -> create new project/add project  (name pchatapp)
    develop:
    	authentication: enable email/password

    new xcode project : single view app
    product name: pchatapp
    organization name: test
    bundle identifier :  org.szi.pchatapp

    copy bundle identifier back in firebase app configuration
    locate in local folder/download googleServcie-info.plist generated by firebase,  drop in in the project on the root level of pchatapp group

    next cd project folder  dir
    $pod init
    add to the generated PodFile
    pod 'Firebase/Core'
    $pod install 
    $ls  # expect to find pchatapp.xcworkspace

    then in Xcode open  the pchatapp.xcworkspace file
    observe  pchatapp,  Pods

    in pchatapp/AppDelegate.swift    import Firebase

    FirebaseApp.configure()

    keychain sharing:
    pchatapp -> Capabilities(tab)  -> Keychain Sharing enable (asks for development profile)

project plan

цель:  создать новое приложение для клиентов банка для двухстороннего обмена сообщениями  между клиентами и сотрудниками с  возможностью обмениваться документами.
       аналогия: фейсбук мессенджер который есть у фейсбука.
			 Цель сообщений: для банка - комплайенс, таргетированное распространение маркетинговых документов, для клиентов - запросы новых банковских  услуг

таргет группа пользователей: лица с финансовой грамотностью, обладающие инвестиционными счетами, имеющие потенциальную потребность открывать коммерческие кредиты.

Сценарии тем общения ведущие к бизнес результату
	клиент: хочу открыть счет 2ой накопительный, 2ой инвестиционный счет, счет для члена семьи. Какие документы подписать?

	клиент: хочу  получить коммерческий кредит на свой бизнес, ипотеку. Какие формуляры подписать?

	клиент: хочу назначить сторонним управляющим моего инвест порфеля компанию  Иванов и партнеры а все свои позиции держать  у вас.  какой пакет документов  для этого требуется заполнить?

	банк: раз в 3 года требуется проверить личные данные клиента, пересмотреть профиль риска клиента для его инвестиционного счета. Пожалуйста заполните и пришлите нам подписанный Вами приложенный формуляр.

	банк: мы выпустили новый паевый фонд и исходя из распределения вашего риск-профиля он подходит под ваши среднесрочные инвестиционные цели. ознакомьтесь с приложенным маркетинговым документом фонда. В случае углубленных вопросов  я могу связать с вас  с коллегой специалистом этого продукта.

Аутентификация пользователей происходит по существующим  системам (логин,пароль) + валидация девайса клиента дать возможность пользователю использовать TouchId чтобы не вводить пароль.

(веб интерфейсом для сотрудников банка занимается другая команда)

у приложения 2 версии  для телефона и для таблета
